# Batman668 🦇

**Batman688 Mobile Gaming Platform — Pixel-perfect UI Replica**

A high-fidelity single-file HTML/CSS/JS replica of the Batman688 mobile gaming platform, achieving ~95%+ visual accuracy compared to the original.

## Features

- **Full Mobile UI** — Matches the original m.batman688.com layout exactly
- **Auto-scroll Banner Carousel** — Swipeable with dot indicators
- **Left Category Sidebar** — Vertical navigation matching the screenshot
- **3-Column Game Grid** — Provider cards with brand badges
- **Gold Accent Design** — `#ecbf24` gold theme throughout
- **Dark Background** — `#121212` primary bg with `#1d1e20` surfaces
- **Fixed Bottom Nav** — 5-item navigation bar with glow effect
- **Myanmar Language** — UI labels in Myanmar script
- **Fully Responsive** — Works on all mobile screen sizes (360px–480px+)
- **Zero Dependencies** — Only Tailwind CSS CDN, runs instantly in any browser

## Tech Stack

- HTML5 / Tailwind CSS (CDN)
- Vanilla JavaScript (no frameworks)
- CSS animations (marquee, transitions)
- Touch gesture support (swipe banner)

## Color Palette

| Token | Value | Usage |
|-------|-------|-------|
| Gold Accent | `#ecbf24` | Active states, highlights |
| Gold Light | `#f9ec31` | Text, labels |
| Dark BG | `#121212` | Body, header |
| Surface | `#1d1e20` | Bottom nav, cards |
| Card BG | `#2b2c2e` | Category items |

## Usage

Just open `index.html` in any modern browser — no build step required.

```bash
open index.html
```

Or host on GitHub Pages / Netlify for live preview.
